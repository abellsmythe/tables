Default table style

By default tables are styled with inner horizontal borders. Table headings and data are set at 13px. No additional classes necessary.

# First Name Last Name Language
1 Some One English
2 Joe Sixpack English
3 Stu Dent Code 
            
                <table>
                    <thead>
                        <tr>
                            <th>#</th>
                            <th>First Name</th>
                            <th>Last Name</th>
                            <th>Language</th>
                        </tr>
                    </thead>
                    <tbody>
                        <tr>
                            <td>1</td>
                            <td>Some</td>
                            <td>One</td>
                            <td>English</td>
                        </tr>
                        <tr>
                            <td>2</td>
                            <td>Joe</td>
                            <td>Sixpack</td>
                            <td>English</td>
                        </tr>
                        <tr>
                            <td>3</td>
                            <td>Stu</td>
                            <td>Dent</td>
                            <td>Code</td>
                        </tr>
                    </tbody>
                </table>

Condensed table style

For large tables where tight spacing is required. Just add the class condensed

# First Name Last Name Language
1 Some One English
2 Joe Sixpack English
3 Stu Dent Code 
            
                <table class="condensed">
                    <thead>
                        <tr>
                            <th>#</th>
                            <th>First Name</th>
                            <th>Last Name</th>
                            <th>Language</th>
                        </tr>
